# Primer Parcial de Métodos Numéricos Avanzados — Tema II

## Ejercicio 1

Dada la Transformación Lineal $T : \mathbb{R}^3 \to \mathbb{R}^3 : T(x, y, z) = (x - y + z,\ x + 3y - z,\ z + y - 2x)$

a) Hallar el Núcleo y la Imagen
b) Hallar $(x, y, z)$, si existe, de modo que $T(x, y, z) = (2, 2, -2)$

## Ejercicio 2

Considere la matriz

$$A = \begin{pmatrix} 0.55 & 1.1 \\ 1.1 & 0.8 \end{pmatrix}$$

a) Realizar la descomposición en valores singulares.
b) Realizar la factorización QR.

## Ejercicio 3

Dado el siguiente conjunto de datos

| x | y |
|---|---|
| $0$ | $0.78$ |
| $1$ | $1.9$ |
| $2$ | $6.7$ |

Usando cuadrados mínimos, realice el ajuste de los datos experimentales a la ecuación

$$y(x) = a x^2 + b$$

Utilice descomposición QR para resolver el problema de cuadrados mínimos.
