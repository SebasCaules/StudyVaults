# 2026-04-16 — Transformaciones Lineales: Inyectividad, Sobreyectividad y Teorema de las Dimensiones

> Anotaciones manuscritas. La clase arranca a media charla (sin definición formal de TL en este PDF — se asume vista) y desarrolla: caracterización de inyectividad por núcleo, sobreyectividad, biyectividad, teorema de las dimensiones, definición de TL por imagen sobre una base, y varios ejercicios.

## Caracterización de inyectividad

Sea $T: V \to W$ transformación lineal. **$T$ es inyectiva $\iff N(T) = \{0\}$.**

**Demostración del $(\Leftarrow)$:**

$$T(v_1) = T(v_2) \implies T(v_1) - T(v_2) = 0 \implies T(v_1 - v_2) = 0 \implies v_1 - v_2 \in N(T) \implies v_1 - v_2 = 0 \implies v_1 = v_2$$

## Sobreyectividad

**2) $T$ es sobreyectiva si $W = R(T)$.**

Equivalentemente:

$$\forall w \in W, \exists v \in V : w = T(v)$$

## Biyectividad

**3) $T$ es biyectiva si es inyectiva y sobreyectiva.**

## Teorema de las dimensiones

Sean $V, W$ espacios vectoriales de dimensión finita y $T: V \to W$ una transformación lineal. Entonces

$$\dim(V) = \dim(N(T)) + \dim(R(T))$$

### Corolario

Si $\dim(V) = \dim(W)$ entonces $T$ es inyectiva $\iff T$ es sobreyectiva.

## Algunos comentarios informales sobre $T: V \to W$

i) $\forall v \in V, \exists w \in W : w = T(v)$
ii) Si $w_1 = T(v)$ y $w_2 = T(v) \implies w_1 = w_2$

Si tengo definida la transformación sobre una base ocurre (i): $B = \{v_1, \dots, v_n\}$ base, $T(v_1) = w_1, \dots, T(v_n) = w_n$.

$v \in V$, $v = \sum_{k=1}^{n} \alpha_k v_k \implies T(v) = \sum_{k=1}^{n} \alpha_k T(v_k)$.

Pero ojo: si tengo $\{v_1, v_2\}$ y un subconjunto $S = \{v_1\}$, no existe imagen para $v_2$ — la definición debe ser sobre una base completa.

## Teorema (definición de TL por sus valores en una base)

Sean $V, W$ espacios vectoriales, $B = \{v_1, \dots, v_n\}$ base de $V$ y $\{w_1, \dots, w_n\} \subseteq W$.

$$\implies \text{Existe una única TL que cumple } T(v_i) = w_i \quad 1 \le i \le n$$

## Ejemplos / Ejercicios

### 1) $T: \mathbb{R}^{n \times m} \to \mathbb{R} : T(A) = \rho(A)$

El **rango** de una matriz es la cantidad de filas (o columnas) que son LI.

Veamos si es TL. Contraejemplo a la aditividad:

$$A = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}, \quad B = \begin{pmatrix} 1 & 1 \\ 0 & -1 \end{pmatrix}$$

$\rho(A) + \rho(B) = 2 + 2 = 4$ (cada una tiene rango 2).

Pero

$$A + B = \begin{pmatrix} 2 & 2 \\ 0 & 0 \end{pmatrix}, \quad \rho(A + B) = 1$$

Como $\rho(A + B) \neq \rho(A) + \rho(B)$, **$T$ no es TL.**

### 2) $T: \mathbb{R}^{n \times n} \to \mathbb{R}^{n \times n} : T(A) = A + A^T$

i) $T(A + B) = (A + B) + (A + B)^T = A + B + A^T + B^T = (A + A^T) + (B + B^T) = T(A) + T(B)$ ✓
ii) $T(\lambda A) = \lambda A + (\lambda A)^T = \lambda A + \lambda A^T = \lambda (A + A^T) = \lambda T(A)$ ✓

**Es TL.**

**Núcleo:** $T(A) = 0 \implies A + A^T = 0 \implies A = -A^T$. El núcleo son las **matrices antisimétricas**.

**Imagen:** Pensarlo (son las matrices simétricas, ya que $A + A^T$ siempre es simétrica).

### Ejercicio: existencia y unicidad de TL

Sea $C = \{v_1, v_2, v_3\} \subseteq \mathbb{R}^3$. Analizar la existencia y unicidad de una TL que cumpla

$$T(v_1) = (1, 1, 1), \quad T(v_2) = (0, -1, 3), \quad T(v_3) = (2, 5, -7)$$

en cada uno de los siguientes casos:

**a)** $v_1 = (1, 1, 1), v_2 = (0, 1, 0), v_3 = (2, -1, 2)$

i) Veamos si $\{v_1, v_2, v_3\}$ es base. ¿$v_3$ es CL de $v_1, v_2$?

$$(2, -1, 2) = \alpha (1, 1, 1) + \beta (0, 1, 0)$$

Sistema: $\alpha = 2$, $\alpha + \beta = -1 \implies \beta = -3$. Tercera coordenada: $\alpha = 2$ ✓. **No es base** (es LD).

Como $(2, -1, 2) = 2(1, 1, 1) - 3(0, 1, 0)$, se requiere por linealidad

$$T(2, -1, 2) = 2 T(1, 1, 1) - 3 T(0, 1, 0)$$

Chequeo: $2(1, 1, 1) - 3(0, -1, 3) = (2, 2, 2) - (0, -3, 9) = (2, 5, -7)$ ✓ (coincide con el dado).

**Existe pero no es única** (se puede extender $T$ libremente sobre el complemento; ej. $T(\tilde v_3) = w_*$ arbitrario para cualquier $\tilde v_3$ que complete a una base).

**b)** $v_1 = (1, 1, 1), v_2 = (0, 1, 0), v_3 = (1, 0, 1)$

Observación: $v_3 = v_1 - v_2$. Para que exista la TL debe cumplirse $T(v_3) = T(v_1) - T(v_2)$.

$T(v_1) - T(v_2) = (1, 1, 1) - (0, -1, 3) = (1, 2, -2)$

Pero el dato dice $T(v_3) = (2, 5, -7)$. Como $(1, 2, -2) \neq (2, 5, -7)$, **no existe** la TL.

**c)** $v_1 = (1, 0, 3), v_2 = (0, 1, 1), v_3 = (1, 0, 0)$

Verifico si son LI calculando el determinante (matriz por filas):

$$\begin{vmatrix} 1 & 0 & 0 \\ 1 & 0 & 3 \\ 0 & 1 & 1 \end{vmatrix} = 1 \cdot (0 \cdot 1 - 3 \cdot 1) - 0 + 0 = -3 \neq 0$$

Son LI, son base. Por el teorema **existe y es única.**
