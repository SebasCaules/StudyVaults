# Guía de Trabajos Prácticos IX — Transformada Continua de Fourier

## Ejercicio 1

Hallar la transformada de Fourier, por integración directa de las siguientes funciones:

**a)** $x(t) = \begin{cases} e^{-at} & \text{si } t \ge 0 \\ 0 & \text{si } t < 0 \end{cases}$

**b)** $x(t) = \begin{cases} 1 & \text{si } |t| \le \tfrac{a}{2} \\ 0 & \text{si } |t| \ge \tfrac{a}{2} \end{cases}$

**c)** $\Lambda_n(t) = \begin{cases} \dfrac{n + t}{n^2} & \text{si } -n < t < 0 \\ \dfrac{n - t}{n^2} & \text{si } 0 < t < n \end{cases}$

**d)** $x(t) = e^{i\alpha x}\, e^{\beta x}$

**e)** $x(t) = A\, e^{-\alpha |t|}$ con $A \in \mathbb{R}$ y $\alpha \in \mathbb{R}_{>0}$

En cada caso graficar su módulo y fase.
