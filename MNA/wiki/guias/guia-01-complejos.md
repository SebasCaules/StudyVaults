# Guía de Trabajos Prácticos I — Números Complejos

## Ejercicio 1

Dados los siguientes números complejos

$$z_1 = i,\quad z_2 = 1 + i,\quad z_3 = -1,\quad z_4 = 81,\quad z_5 = -21$$

**a)** Expresarlos en forma polar exponencial y polar trigonométrica.

**b)** Hallar el resultado de cada una de las siguientes operaciones:

- i) $z = z_2\, z_5\, z_1^{-1}$
- ii) $p = z_1^3\, z_2^3$
- iii) $q = \dfrac{z_1 - z_3}{z_2^2}$
- iv) $r = z_1^{245}$
- v) $s = z_1^{z_2}$
- vi) $t = z_4^{z_2}$

**c)** Resolver las siguientes ecuaciones:

- i) $w^4 = z_4$
- ii) $w^6 = z_5$
- iii) $e^w = z_1$
- iv) $e^w = z_2\, z_5\, z_1^{-1}$
- v) $w^4 = z_2$
- vi) $e^w = z_3$
- vii) $e^w = z_5\, q \;\wedge\; q^4 = z_1^{-1}$

## Ejercicio 2

Resolver las siguientes ecuaciones en $\mathbb{C}$ donde $z = x + i\,y$ con $x, y \in \mathbb{R}$.

- i) $2x + 3i = -5 + iy$
- ii) $-4 + 6yi = \dfrac{1}{2} x + \sqrt{3}\, y\, i$
- iii) $i(x - iy) = x + 5 + 2yi$
- iv) $(1 - 2i)x + (3 + 5i)y = 1 + 3i$
- v) $x + 2y + (4x - 3y)i = 2x - 1 + (y - 6)i$

## Ejercicio 3

Escribir en forma binómica los siguientes números complejos, donde $a, b, c, d, \alpha \in \mathbb{R}$.

- i) $z = \dfrac{1 + i\,\tan\alpha}{1 - i\,\tan\alpha}$
- ii) $z = \dfrac{a + ib}{c + id} + \dfrac{a - ib}{c - id}$
- iii) $z = \dfrac{a - ib}{(a + ib)^2}$

## Ejercicio 4

Resolver las siguientes ecuaciones:

- i) $z^n = \overline{z}$, con $n \in \mathbb{N}$
- ii) $z^n = i$, con $n \in \mathbb{N}$

## Ejercicio 5

Sea $z = x + iy$ con $x, y \in \mathbb{R}$ se pide hallar:

- i) $\operatorname{Im}\left(\dfrac{1}{1+i}\right)$
- ii) $\operatorname{Re}\left(\dfrac{z}{\overline{z}}\right)$
- iii) $\operatorname{Re}\left(\dfrac{(1-i)^2}{4+2i}\right)$
- iv) $\operatorname{Im}(z^3)$
- v) $\operatorname{Re}\left(\dfrac{1}{z^2}\right)$
- vi) $\dfrac{-z}{\overline{z}}$

## Ejercicio 6

Escribir los siguientes números complejos en forma polar:

- i) $z = \sqrt{3} + i$
- ii) $z = -2 - 2\sqrt{3}\, i$
- iii) $z = -5$
- iv) $z = -3i$
- v) $z = \dfrac{-2}{1 + i\sqrt{3}}$

## Ejercicio 7

Escribir las siguientes expresiones como $z = a + ib$ con $a, b \in \mathbb{R}$:

- i) $z = \dfrac{(-1 + \sqrt{3}\,i)^{15}}{(1 - i)^{20}} + \dfrac{(-1 - \sqrt{3}\,i)^{15}}{(1 + i)^{20}}$
- ii) $z = \dfrac{(1 + i)^n}{(1 - i)^{n-1}}$, $n \in \mathbb{N} - \{1\}$
- iii) $z = \left(\dfrac{1 + \sqrt{3}\,i}{1 - i}\right)^{17}$
- iv) $z = \left(-\dfrac{1}{2} + \dfrac{\sqrt{3}}{2}\, i\right)^n$, $n \in \mathbb{N}$
- v) $z = (\sqrt{3} - i)^5$

## Ejercicio 8

Hallar $n \in \mathbb{N}$ de modo que

$$(\sqrt{3} - i)^n = 2^{n-1}(-1 + i\sqrt{3})$$

## Ejercicio 9

Resolver las siguientes ecuaciones y representar gráficamente la solución:

- a) $|z| - z = 1 + 2i$
- b) $z^4 + 1 = 0$
- c) $z + \overline{z} = 4$
- d) $(1 + i)\, z = 1$
- e) $\dfrac{1}{z} = i$
- f) $z^2 = 3 + 4i$
- g) $z^2 + (2 + i)\, z + (-1 + 7i) = 0$
- h) $z^2 = (1 - i)^2$
- i) $z^4 = -\dfrac{1}{2} + i\dfrac{\sqrt{3}}{2}\,\overline{z}^4$
- j) $z^6 + 1 = \sqrt{3}\, i$
- k) $z^2 + |z^2| = i\, z$
- l) $|z|^2 = (z + \overline{z})\,\operatorname{Im}(z)$
- m) $z^4 = i\, \overline{z}^3$
- n) $z^8 = \overline{z}^8$
- o) $z^{12} + z^6 + 1 = 0$
- p) $z^6 = (2 - 2i)^{10}$
- q) $(z - 1)^4 = (z + i)^4$
- r) $(z + 1)^4 = (z + i)^4$
- s) $3 z^{12} + 5|z|^{12} - 8 = 0$
