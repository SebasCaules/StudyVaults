## Definición

**Estado (cuadro) de Resultados:** descripción de los **hechos y factores que generaron el resultado** atribuible a un período. Computa solo resultados **del período**, **devengados** ("ganado o perdido", que no necesariamente es lo cobrado o pagado — [[devengado-vs-percibido]]), clasificados en ordinarios y extraordinarios.

## Esquemas

**Elemental:**

| | |
|---|---|
| Ingresos (ventas / servicios) | |
| − Costos del negocio | |
| **Utilidad operativa** | |
| − Intereses | |
| ± Resultados de inversiones | |
| ± Resultados extraordinarios | |
| **Utilidad antes de IG** | |
| − Impuesto a las ganancias | |
| **Utilidad neta del ejercicio** | |

**Financiero** (cascada [[ebit-ebitda]]):

$$Ventas - Costos\ erogables = EBITDA$$
$$EBITDA - Amortizaciones = EBIT$$
$$EBIT - Intereses = EBT$$
$$EBT - IG = EAT$$

**Donde:**
- $EBITDA$: utilidad antes de intereses, impuestos y amortizaciones
- $EBIT$: utilidad operativa (antes de intereses e impuestos)
- $EBT$: utilidad antes de impuestos
- $EAT$: utilidad neta (después de impuestos)
- $IG$: impuesto a las ganancias

## Mecánica contable

Las cuentas de resultado son **transitorias**: Ingresos (R+: ventas, alquileres ganados, intereses ganados, venta de bienes de uso) y Egresos (R−: sueldos, impuestos, amortizaciones, costo de venta, servicios). Al cierre se "vacían" contra la cuenta **Utilidades** (asiento de cierre): si queda saldo acreedor hubo ganancia; si deudor, pérdida. El resultado pasa al PN del [[balance-patrimonial]] como Resultado del ejercicio.

## Ejemplo

CONTA, marzo/19 (GP5 Ej. 1, valuación PEPS, en \$):

| | |
|---|---|
| Ventas (100 u × \$4) | 400 |
| − CMV (100 u × \$2,75) | (275) |
| **Utilidad bruta** | 125 |
| − Sueldos y cargas (100 + 30%) | (130) |
| − Alquileres, energía | (60) |
| − Amortizaciones | (80) |
| **Utilidad operativa** | (145) |
| + Resultado venta de bonos (500 − 400) | 100 |
| **Resultado del ejercicio** | **(45)** |

Notar: la venta de bonos **no** es "Ventas" (no es el objeto del negocio); entra como resultado de inversiones.

## Errores comunes / Distinciones

- **No es un estado de caja.** Ventas a crédito suman; cobranzas de períodos anteriores no.
- **La amortización resta aunque no se pague a nadie** — es la cuota devengada del activo fijo.
- **Solo resultados del período:** pagar sueldos adeudados de febrero en marzo no es gasto de marzo.
- **Utilidad neta positiva no garantiza solvencia:** ver [[estado-origen-aplicacion-fondos]].
